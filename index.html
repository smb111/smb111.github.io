交叉编译 jrtplib-3.11.1 和 jthread-1.3.3

发表于 2017-08-04   |   分类于 video

就不吐槽网上的无用和误导信息了

下载 jrtplib 和 jthread

http://research.edm.uhasselt.be/jori/page/CS/Jrtplib.html

放入 ubuntu 中并进行解压

因为 jrtplib 依赖 jthread 的库，所以我们先编译 jthread 库

  1. 进入 jthread 的解压目录，执行 cmake . (必须确保cmake的版本高于3.0) 升级或者安装cmake3.2 https://askubuntu.com/questions/610291/how-to-install-cmake-3-2-on-ubuntu-14-04
  2. 修改 CMakeCache.txt 文件 把/usr/bin/ar ld nm objcopy objdump strip cc c++ 一系列 c 以及 c++ 的编译器和连接器 修改成 arm-linux- ar ld nm objcopy objdump strip gcc g++
  3. 执行 make
  4. 执行 make install 这样之后 生成的 库文件 和 头文件 分别在 /usr/local/lib /usr/local/include/jthread
  5. 接下来编译 jrtplib 库，步骤同上面的一样
