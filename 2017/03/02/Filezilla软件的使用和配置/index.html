出现这个警告并没有注意，因为已经和开发板连接，并且可以进行文件传输了

但是传过去的很简单的程序，就是运行不起来，出现 segmentation fault 没有任何其他信息，我程序里面的输出的调试信息也没显示在终端，说明程序没跑
