常量指针和指针常量


1.（指针）和 const（常量） 谁在前先读谁 ；

2.象征着地址，const象征着内容；

3.谁在前面谁就不允许改变。
 从英文翻译上看更明白，常量指针（const pointer）

常量指针 (const pointer) p1：指向的地址可以变，但内容不可以重新赋值，内容的改变只能通过修改地址指向后变换。 

p1 = &a是正确的，但 *p1 = a是错误的。

指针常量 (pointer to const) p2：指向的地址不可以重新赋值，但内容可以改变，必须初始化，地址跟随一生。

p2= &a是错误的，而*p2 = a 是正确的。

指针数组和数组指针的区别及区分

数组指针（也称行指针）数组指针只是一个指针变量

定义: int (*p)[n]; ()优先级高，首先说明p是一个指针，指向一个整型的一维数组,这个一维数组的长度是n，也可以说是p的步长。

也就是说执行p+1时，p要跨过n个整型数据的长度。

指针数组 指针数组是多个指针变量

定义: int *p[n]; []优先级高，先与p结合成为一个数组，再由int*说明这是一个整型指针数组，它有n个指针类型的数组元素。

这里执行p+1是错误的，这样赋值也是错误的：p=a；因为p是个不可知的表示，只存在p[0]、p[1]、p[2]…p[n-1],而且它们分别是指针变量可以用来存放变量地址。但可以这样 p=a; 这里p表示指针数组第一个元素的值，a的首地址的值。

总结

这样两者的区别就豁然开朗了，数组指针只是一个指针变量(行指针)，似乎是C语言里专门用来指向二维数组的，它占有内存中一个指针的存储空间。
