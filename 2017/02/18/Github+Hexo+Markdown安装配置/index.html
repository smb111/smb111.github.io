搭建完成之后的总结

    1. E:\HEXO\source_posts 下面将通过MarkDownPad 2写好的 .md文件放入这个目录中

  • 2.写.md文件的时候开头部分的格式固定

    | title | 所写文章的标题 |
    | ————- |:————-: |
    | date | 显示改文章写作的时间 |
    | description | 关于这篇文章的描述 |
    | categories | 分类（用于创建分类的不同文件夹） |
    | tags | 文章的标签用于分类 |

  • 3.在Hexo的根目录下面打开 Git Bash 输入：hexo -s –debug.在浏览器中输入 localhost:4001 查看效果，如果效果可行，停掉当前的bash，Ctrl+ｃ。　执行 hexo g 生成静态页面，最后同步到github，hexo d。 如果是修改主题之前要清理一下 hexo clean
