C++中的STL算法

发表于 2017-02-27   |   分类于 C++

STL算法简介

STL中的算法大致可以分为以下七类:

  1. 不变序列算法
  2. 变值算法
  3. 删除算法
  4. 变序算法
  5. 排序算法
  6. 有序区间算法
  7. 数值算法
大多重载的算法都是有两个版本的
  1. 用 “==” 判断元素是否相等, 或用 “<” 来比较大小
  2. 多出一个类型参数 “Pred” 和函数形参 “Pred op”

通过表达式 “op(x,y)” 的返回值: ture/false判断x是否 “等于” y，或者x是否 “小于” y

不变序列算法

  1. 该类算法不会修改算法所作用的容器或对象
  2. 适用于顺序容器和关联容器
  3. 时间复杂度都是O(n)

常用不变序列算法

find: 在区间中查找等于某值的元素

返回区间 [first,last) 中的迭代器 i ,使得 * i == val

find_if:在区间中查找符合某条件的元素

返回区间 [first,last) 中的迭代器 i, 使得 pr(*i) == true

for_each:对区间中的每个元素都做某种操作

对[first, last)中的每个元素e, 执行f(e), 要求 f(e)不能改变e

count:计算区间中等于某值的元素个数

计算[first, last) 中等于val的元素个数(x==y为true算等于)

count_if:计算区间中符合某种条件的元素个数

计算[first, last) 中符合pr(e) == true 的元素e的个数

min_element:求区间中的最小值(可自定义比较器)

返回[first,last) 中最小元素的迭代器, 以 “<” 作比较器， 最小指没有元素比它小, 而不是它比别的不同元素都小 因为即便a!= b, a<b 和b<a有可能都不成立

max_element:求区间中的最大值(可自定义比较器)

返回[first,last) 中最大元素(不小于任何其他元素)的迭代器,默认以 “<” 作比较器

变值算法

  1. 此类算法会修改源区间或目标区间元素的值
  2. 值被修改的那个区间, 不可以是属于关联容器的

常用变值算法

copy: 复制一个区间到别处

本函数对每个在区间[0, last - first)中的N执行一次

(x+N) = (first + N), 返回 x + N

对于copy(v.begin(),v.end(),output);

first 和 last 的类型是 vector::const_iterator

output 的类型是 ostream_iterator

transform 将一个区间的元素变形后拷贝到另一个区间

template
OutIt transform(InIt first, InIt last, OutIt x, Unop uop);

对[first,last)中的每个迭代器I

  • 执行 uop( * I ); 并将结果依次放入从 x 开始的地方
  • 要求 uop( I ) 不得改变 I 的值
  • 本模板返回值是个迭代器, 即 x + (last-first), x可以和 first相等

删除算法

删除一个容器里的某些元素

删除 – 不会使容器里的元素减少
1. 将所有应该被删除的元素看做空位子
1. 用留下的元素从后往前移, 依次去填空位子
1. 元素往前移后, 它原来的位置也就算是空位子
1. 也应由后面的留下的元素来填上
1. 最后, 没有被填上的空位子, 维持其原来的值不变
 删除算法不应作用于关联容器

常用删除算法

remove : 删除区间中等于某个值的元素
remove_if : 删除区间中满足某种条件的元素
unique : 删除区间中连续相等的元素, 只留下一个(可自定义比较器(什么是相等可自定义默认使用 == ))返回最后删除的元素的后一个元素的迭代器（和begin相减就会知道还剩多少元素）

对[first,last) 这个序列中连续相等的元素, 只留下第一个

返回值是迭代器, 指向元素删除后的区间的最后一个元素的后面

变序算法

  1. 变序算法改变容器中元素的顺序
  2. 但是不改变元素的值
  3. 变序算法不适用于关联容器
  4. 算法复杂度都是O(n)的

常用变序算法

  1. reverse 颠倒区间的前后次序
  2. next_permutation 将区间改为下一个排列(可自定义比较器) 比如 { 1,2，3 } 的排列， [1,2,3] 排列的下一个排列是 [1,3,2] 下一个 [2,1,3] 下一个 [2,3,1]
  3. prev_permutation 将区间改为上一个排列(可自定义比较器)
  4. random_shuffle 随机打乱区间内元素的顺序（适用于能随机访问的元素）
  5. stable_patition把区间内满足某个条件的元素移到前面,s不满足该条件的移到后面,而对这两部分元素, 分别保持它们原来的先后次序不变

排序算法

  1. 比前面的变序算法复杂度更高, 一般是O(nlog(n))
  2. 排序算法需要随机访问迭代器的支持
  3. 不适用于关联容器和list

常用排序算法

sort 将区间从小到大排序(可自定义比较器：缺省的话使用 < 进行比较)

按升序排序
判断x是否应比y靠前, 就看 x < y 是否为true

有序区间算法

  1. 要求所操作的区间是已经从小到大排好序的
  2. 需要随机访问迭代器的支持
  3. 有序区间算法不能用于关联容器和list

常用有序区间算法

  1. binary_search 判断区间中是否包含某个元素
  2. lower_bound 查找最后一个不小于某值的元素的位置
  3. upper_bound 查找第一个大于某值的元素的位置
  4. equal_range 同时获取lower_bound和upper_bound
  5. merge 合并两个有序区间到第三个区间

binary_search：折半查找

要求容器已经有序且支持随机访问迭代器, 返回是否找到
