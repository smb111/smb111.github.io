ubuntu14.04下更换内核

发表于 2017-07-12   |   分类于 Ubuntu-Linux

升级内核

  1. 显示所有可以获取的内核

    apt-cache search linux

  2. 安装内核，假设你要安装的内核为x.x.x-x，则使用下面的命令

sudo apt-get install linux-headers-x.x.x-x-generic linux-image-x.x.x-x-generic
  1. 安装后，reboot即可，重启后，既是以新内核启动

设置内核启动的顺序

  1. 查看 grub.cfg 中的内核的顺序

    vim /boot/grub/grub.cfg

从menuentry开始看起:

第一个是 Ubuntu，Linux 4.4.0-31-generic
第二个是 Ubuntu, with Linux 4.4.0-31-generic (recovery mode)
第三个是 Ubuntu，Linux 3.13.0-24-generic
第四个是 Ubuntu, with Linux 3.13.0-24-generic (recovery mode)

  1. 修改开机默认启动的 内核版本

    vim /etc/default/grub

修改 GRUB_DEFAULT 的值为2，因为上面看到有grub中有四个内核，按照顺序我们使用 Ubuntu，Linux 3.13.0-24-generic 因为计数从零开始，所以这个地方我们写入2

使我们的配置生效

update-grub 如果有再执行 update-grub2

总之电脑上有的grub版本我们都同步过去！！！
